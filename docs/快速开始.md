# 快速开始

## 下载代码

```bash
git clone https://github.com/citywill/pocket-stack.git
cd pocket-stack
```

## 启动后端 (PocketBase)

**1. 下载 PocketBase 二进制文件**

Pocketbase 下载地址：https://pocketbase.io/docs/

**2. 运行 PocketBase**

```bash
./pocketbase serve
```

**3. 访问并初始化 Pocketbase**

访问地址：`http://127.0.0.1:8090/_/`。第一次访问需要创建管理员账号密码。

>[!warning] 注意
>PocketBase 文件一般存放到项目目录以外。如果想要存放到项目目录中，可以放到 `./private/` 目录下，以免污染git仓库。

## 启动前端

**1. 配置环境变量**

```bash
cp .env.example .env
```
创建 `.env` 文件，根据需要修改其中的变量。

其中 `VITE_POCKETBASE_URL` 为 PocketBase 后端地址。

**2. 安装依赖**

```bash
npm install
```

**3. 启动开发服务器**

```bash
npm run dev
```

默认网址为： `http://localhost:5173`
