# 快速开始

## 启动并初始化 PocketBase 

PocketBase 二进制文件下载地址：https://pocketbase.io/docs/

```bash
./pocketbase serve
```

默认管理界面地址：http://127.0.0.1:8090/_/

首次使用要创建管理员（superuser）账号密码。

## 启动前端

```bash
npm run dev
```

访问 http://localhost:5173

使用 superuser 管理员账号登录。

访问用户管理页面。

创建普通用户。

登出。

使用刚刚创建的普通用户账号登录。
