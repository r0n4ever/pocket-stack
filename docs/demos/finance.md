# 口袋记账

口袋记账是一个基于 PocketBase 的个人财务管理模块，提供记账、分类管理和收支统计功能。

该模块包含三个主要页面：
1.  **账单概览** (`/finance`)：展示本月收支统计和每日趋势图表。
2.  **记账记录** (`/finance/records`)：查看、搜索、筛选和管理收支记录。
3.  **分类管理** (`/finance/categories`)：自定义收支分类。

## 1. 账单概览

访问路径：`/finance`

-   **本月统计卡片**：
    -   本月总收入
    -   本月总支出
    -   本月结余
-   **收支趋势图**：使用面积图（Area Chart）展示本月每日的收入和支出趋势。
-   **自动刷新**：页面加载时自动获取本月数据。

![账单概览](screenshots/finance.png)

## 2. 记账记录

访问路径：`/finance/records`

-   **记录列表**：分页展示收支记录，包含时间、分类、备注、金额等信息。
-   **筛选与搜索**：
    -   支持按备注关键词搜索。
    -   支持按类型（全部/收入/支出）筛选。
-   **新增/编辑记录**：通过侧边抽屉（Drawer）形式快速添加或修改记录。
    -   支持选择收支类型。
    -   支持选择分类（联动类型）。
    -   支持选择日期和填写备注。
-   **删除记录**：支持删除单条记录，删除前有确认弹窗。

![记账记录](screenshots/finance-records.png)

## 3. 分类管理

访问路径：`/finance/categories`

-   **分类列表**：展示所有收支分类，包括名称和类型。
-   **新增/编辑分类**：通过侧边抽屉管理分类信息。
    -   自定义分类名称。
    -   选择分类类型（收入/支出）。
-   **删除分类**：删除不再使用的分类。
